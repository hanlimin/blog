# xargs 

`xargs`命令的作用，是将标准输入转为命令行参数。

- -d 分隔符

    默认情况下，`xargs`将换行符和空格作为分隔符，把标准输入分解成一个个命令行参数。

- -p 询问

    打印出要执行的命令，询问用户是否要执行。

- -t 打印信息

    参数则是打印出最终要执行的命令，然后直接执行，不需要用户确认。

- -0

    使用`null`当作分割符

- -L 

    指定多少行作为命令行参数

- -n

    指定每次将多少项作为命令行参数

- -I

    指定每一项命令行参数的替代字符串。

- --max-procs

    指定同时用多少个进程并行执行命令。
