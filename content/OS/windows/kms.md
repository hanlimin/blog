# KMS 激活

## slmgr

### 常用参数

- upk 卸载产品密钥
- ipk <产品密钥> 安装产品密钥（替换现有密钥）
- ato 激活 Windows
- dli 显示许可证信息
- dlv 显示详细许可证信息
- xpr 当前许可证的截止时间
- ckms 清除KMS服务器名和端口号并设为初始状态

### 高级参数

- skms 设置 KMS 服务器地址
- cpky 从注册表中清除产品密钥（防止泄露引起的攻击）
- ilc <许可证文件> 安装许可证
- rilc 重新安装系统许可证文件
- rearm 重置计算机的授权状态
- dti 显示安装 ID 以进行脱机安装
- atp <确认 ID> 使用电话方式提供的确认 ID 激活产品
